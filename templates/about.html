About Meowcake

Meowcake is a simple website built with Rust and Cloudflare Workers using the Askama templating engine.

The goal of this project is to demonstrate how to build a web application using Rust and Cloudflare Workers, and to provide a simple example of how to use the Askama templating engine.

The source code for Meowcake is available on GitHub.
